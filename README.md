# Архитектура

1. драйвер микроконтроллера из одно файла MAIN.INO
2. Интерфейс приложения на телефон:
   Главное окно
   Выбор файлов из хранилища и галереи на отдельные кнопки
   Прогресс бар отправки
   Недавно отправленные
   Настройки (кастомизация пути загрузки, фоновая работа)
   Меню коннекта устройств

Основная инфа:
Передача файлов по AirDrop и точке доступа wifi
Приблизительное энергопотребление: 600мач ssd в активном состоянии, wifi модуль 80-150мач, плата 150мач = ~900мач в активном состоянии
