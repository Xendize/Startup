## 1. Сегмент + проблема + цель MVP
### Целевой сегмент:
Мобильные креаторы (фрилансеры-фотографы, видеографы), работающие на выезде без ноутбука.

### Ключевая проблема и сценарий:
Нет быстрого беспроводного доступа к большим файлам (видео 4K+) на SSD с телефона/ПК; кабели неудобны, облако требует интернета.

### Гипотеза ценности:
Креаторы готовы использовать портативное SSD с Wi-Fi передачей, если скорость >50 МБ/с и автономия 6+ часов, т.к. это ускорит workflow на 70%.

### Цель MVP:
Проверить, что представители сегмента (креаторы) в сценарии съёмок готовы скачать 5+ ГБ файлов беспроводно за <2 мин и оставить отзыв, потому что это заменяет USB-кабели без потери скорости.

## 2. Сценарий (user flow)
Основной сценарий: Беспроводная передача файлов с SSD на смартфон.

Триггер: Съёмка завершена, файлы на SSD по USB от камеры.

Вход: Включает SSD кнопкой (Wi-Fi сеть активируется).

Подключение: Смартфон подключается к Wi-Fi SSD через app (QR или пароль).

Просмотр: В app видит файлы, фильтрует по типу (MP4), выбирает 5 ГБ.

Передача: Запускает скачивание (прогресс-бар, скорость >50 МБ/с).

Результат: Файлы на смартфоне, checksum OK; подтверждает успех в app.

Фиксация: Оставляет отзыв/рейтинг в app (для метрики).


## 3. MVP Core/Support/Later
### Core:

Автоматическая Wi-Fi точка доступа — шаг 3; проверяет гипотезу скорости передачи.

App для просмотра/скачивания — шаги 4–5; доставляет ценность доступа.

Прогресс-бар + checksum — шаг 6; подтверждает надёжность.

Включение/выключение — шаг 2; базовый контроль.

Браузерный доступ — шаг 3 (ПК); универсальность.

Авто-спящий режим — шаг 7; автономия.

## 5. ТЗ на MVP
### 1. Название и цель:
Беспроводной портативный SSD — MVP для проверки беспроводной передачи файлов с портативного SSD.

### 2. Аудитория и сценарий:
Создатели контента на выезде; см. раздел 2.

### 3. Описание:
Компактное SSD (1 ТБ) с Wi-Fi 6, приложением и веб-доступом для передачи на телефон/ПК без кабелей. Автономия 6 ч, скорость 50+ МБ/с.

### 4. Функциональные требования (Core, user stories):

C1: Как креатор, я хочу включить SSD (или приложить к MagSafe), чтобы активировать Wi-Fi.

C2: Как пользователь, я хочу подключиться и скачать файлы через app.

C3: Как пользователь, я хочу checksum для проверки целостности файлов.

### 5. Критерии приемки (для Core):

Wi-Fi: подключение <10 сек, скорость >50 МБ/с.

Приложение: скачивание 5 ГБ <2 мин, 100% checksum.

Батарея: 6 ч на 50% нагрузке.

### 6. Нефункциональные:

Скорость передачи данных: >50 МБ/с.

Вес: <200 г.

Тип сети: WPA3.

Совместимость: iOS/Android/браузеры.

Защита устройства: IP54.

### 7. Ограничения/допущения:

Нет шифрования (Later).

1 клиент за раз.

Допуск: стабильный Wi-Fi без помех.

### 8. Definition of Done:

Сценарий проходит 95% тестов.

10 бета-тестеров подтверждают ценность.

### 9. Риски:

Wi-Fi помехи (мит: fallback).

Стоимость чипов (+20%).
